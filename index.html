Non-blocking Code

The fibonacci function is not done in the main thread. It is ran in a different thread. If you have watch javascript videos, you might think that you would use async await. But you would be wrong. async await only works for native javascript function that are taken from the javascript engine and sent to the browser to run. Ex: setTimeout, setInterval, fetch, Promise... You might think that because you are using async await that your function returns a Promise, and this means that you code is done asynchronously. But because it's a function that you made it will still be ran by the javascript engine. Blocking you site from working. If you want a normal function to returns asynchronously you will have to use web workers. Web Workers
