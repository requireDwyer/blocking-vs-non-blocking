Non-blocking Code

This Fibonacci function runs in a separate thread, not the main one. If you've watched videos about JavaScript, you might think that using async/await would make it run smoothly. But that's not quite right. async/await only works for built-in JavaScript functions that are sent from the JavaScript engine to the browser to be executed, such as setTimeout, setInterval, fetch, and Promise. So even if you use async/await with your own function, it will still be run by the JavaScript engine, which could cause your website to stop working. To make your function run asynchronously without causing any problems, you should use web workers instead. (check out)
